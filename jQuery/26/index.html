PICT B.E. Course

Course Structure

First Year (FE)

  • Engineering Mathematics - Basic math concepts
  • Physics - Fundamentals of physics
  • C Programming - Introduction to coding
  • Basic Electronics - Electronic components

Second Year (SE)

  • Data Structures - Algorithms and structures
  • OOP - Object-oriented programming
  • DBMS - Database management
  • Computer Networks - Networking basics

Third Year (TE)

  • Operating Systems - Process and memory management
  • Software Engineering - Development lifecycle
  • Web Technologies - HTML, CSS, JS
  • Machine Learning - AI basics

Final Year (BE)

  • Cloud Computing - Cloud services
  • Big Data - Data analytics
  • Cyber Security - Security concepts
  • Project Work - Final year project
